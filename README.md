
# CoffeShop

![CoffeeShope](images/Screenshot%202026-04-24%20192512.png)

"This is a portfolio website template for a cafe, designed and implemented using the modern CSS framework **Tailwind CSS**. The project focuses on showcasing a responsive design and delivering a clean, modern user experience."

---

## 🚀 Live Demo

https://vahidbodaghi.github.io/CoffeeShop/

## 🚀 Features


*   Fully Responsive Design: Excellent display across all devices (desktop, tablet, mobile).
*   Attractive Homepage with a slider/hero section.
*   About Us section detailing the cafe's story.
*   Menu display (food and drinks).
*   Image Gallery.
*   Contact Form.
*   Subtle animations using Tailwind CSS utilities.

## 🛠️ Tech Stack

*   HTML5
*   CSS3
*   Tailwind CSS
*   JavaScript (if used)
